商城網站學習 - Apache x PHP x Hello World

16 May 2019

接續上一篇
商城網站學習 - PHP 學習記錄 1
接著學習PHP與Apache的整合
一開始預設的Apache不知道PHP的位置
也沒辦法使用 PHP提供的 SAPI 跟解析PHP
所以要先在Apache的設定檔中告知PHP的路徑跟初始化位置
續商城網站學習 - Apache 學習記錄 1
Apache安裝路徑為C:\Net\Apache
找到conf的資料夾打開裡面名為httpd.conf的檔案
找到一排LoadModule的地方新增以下資料:

  • LoadModule php7_module "c:/Net/PHP/php7apache2_4.dll"
    加入PHP的module, WINDOW預設的\不能用, 要用/
  • AddHandler php7-script .php
    加入script附檔名為.php的, 注意 .php 前面有空一格
  • PHPIniDir "c:/Net/PHP/"
    PHP初始化的位置

寫一個PHP的網頁試試看 隨便開一個.txt文字檔案
輸入:

<?php
    echo('Hello World');
?>

儲存後改名成 index.php
丟到Apache的預設網頁路徑 c:\Net\Apache\Apache24\htdocs 底下


並且把 該路徑嚇的 index.html 刪除

打開終端機,切換路徑到Apache安裝路徑c:\Net\Apache\Apache24底下
找到bin資料夾 然後輸入httpd啟動Apache


接著打開Chrome, 輸入127.0.0.1:5000
如果看到Hello World, 就是成功了
